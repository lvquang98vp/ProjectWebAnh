# Trang web upload ảnh trực tuyến
## Mục tiêu
```
Xây dựng trang web cho phép upload và lưu trữ ảnh trực tuyến.
```
## Công nghệ sử dụng
```
Back-end: NodeJS
Database: Mongoosedb
Front-end: React native
```
## Hướng dẫn cài đặt
```
Khởi chạy server: Sử dụng lệnh npm start
```
